# 部費管理サイト 💰

サークルや部活動の部費徴収をリアルタイムで共有・管理するためのWebアプリケーションです。

## ✨ 主な機能

* **リアルタイムデータ同期**: 全員の画面がリアルタイムで同期されます。
* **月ごとの記録管理**: ドロップダウンで月を選択し、過去の記録を閲覧・編集できます。月が替わると自動的に画面も更新されます。
* **項目（徴収内容）の管理**: 徴収項目（例：「合宿費」「部費」）を動的に追加・削除・名前変更できます。
* **名簿管理**: 項目ごとに支払った人の名前を改行で入力できます。
* **自動集計**: 項目ごとの「人数」「小計」および「総合計金額」を自動で計算します。
* **過去の記録の保護**: 今月以外の月は「読み取り専用」となり、「過去の記録を編集」ボタンを押すことで編集が解除されます。
* **同期ステータス表示**: サーバーとの接続状態が右上のインジケーター（●）でわかります。

## 🖥️ 使い方

1.  **月の選択**: 上部のドロップダウンで、閲覧または編集したい月（例：「2025-11」）を選択します。
2.  **今月の編集**: 今月（最新の月）は、自由に項目、金額、名前を編集できます。データは入力の約1秒後に自動で保存されます。
3.  **過去の月の編集**: 過去の月を表示すると、入力欄が無効化されます。「過去の記録を編集」ボタンを押すと、ロックが解除され編集可能になります。
4.  **項目の管理**: 各カードの右上にある [...] ボタンから、項目の名前変更や削除ができます。
5.  **項目の追加**: 画面下部の「項目を追加」ボタンから、新しい徴収項目を作成できます。
