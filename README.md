# 部費管理サイト

サークルや部活動の部費徴収をリアルタイムで共有・管理するためのWebアプリケーションです。

## 主な機能

- **リアルタイムデータ同期**: Firebase Realtime Database を使用し、全員の画面がリアルタイムで同期されます
- **月ごとの記録管理**: ドロップダウンで月を選択して過去の記録を閲覧・編集できます。新しい月には直前月の項目構造が自動で引き継がれます
- **収入項目の管理**: 徴収項目（例：部費、合宿費）を動的に追加・削除・名前変更できます
- **支払済み名簿**: 項目ごとに支払った人の名前を管理し、人数・小計を自動集計します
- **支出管理**: 使ったお金を項目名と金額で記録できます
- **過去データの保護**: 今月以外は読み取り専用となり、専用ボタンで編集ロックを解除できます

## プロジェクト構成

```
club-fee-manager/
├── index.html        # メイン画面（HTMLのみ）
├── usage.html        # 使い方ページ
└── js/
    ├── main.js       # エントリーポイント・イベント処理
    ├── state.js      # アプリ全体の状態管理
    ├── firebase.js   # Firebase認証・データ読み書き
    ├── data.js       # データロジック（月管理・集計）
    ├── render.js     # DOM描画
    └── utils.js      # ユーティリティ関数
```

## セキュリティ

- **Firebase匿名認証**: アプリを開くと自動で匿名ログインが行われ、アプリ外からのデータベースへの直接アクセスを防止します
- **セキュリティルール**: `auth != null` のリクエストのみ読み書きを許可しています
- **XSS対策**: Firebaseから取得したデータはHTMLエスケープしてから描画しています

## 使い方

1. **月の選択**: 上部のドロップダウンで閲覧・編集したい月を選択します
2. **今月の編集**: 項目・金額・名前を自由に編集できます。入力の約1秒後に自動保存されます
3. **過去の月の編集**: 「過去の記録を編集」ボタンで編集ロックを解除できます
4. **項目の管理**: 各カード右上の `...` ボタンから項目名の変更・削除ができます
5. **支払済み登録**: 名前を入力して「追加」ボタン（またはEnterキー）で名簿に追加します

## ローカルで動かす

ビルド不要です。ローカルサーバーを起動してブラウザで開くだけで動作します。

```bash
# Python 3
python3 -m http.server 8080
# → http://localhost:8080 をブラウザで開く
```
